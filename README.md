#  Spotify Clone

A Spotify-inspired music player built using **HTML, CSS, and JavaScript**.

## Features
- Dynamic playlist loading
- Audio playback (play / pause / next / previous)
- Custom seek bar & volume control
- Responsive card-based UI using CSS Grid
- Folder-based song management

## Tech Stack
- HTML
- CSS (Grid & Flexbox)
- JavaScript (DOM, Audio API, Fetch)

## Learning Outcome
This project helped me understand real-world frontend development, debugging layout issues, and managing application state.

## Future Improvements
- Convert to full-stack using Node.js
- Backend APIs for playlists
- User authentication
